# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - Initial Archival Release

### Added
- Console-based 2D maze game using ncurses
- Quadruply linked list for maze representation
- Stack-based move history with undo functionality
- Queue-based item collection tracking
- Three difficulty levels (Easy, Medium, Hard)
- Proximity sensing hints for navigation
- Coin collection system with dynamic respawning
- Bomb obstacles with proximity-based visibility
- Score calculation based on coins and remaining moves
